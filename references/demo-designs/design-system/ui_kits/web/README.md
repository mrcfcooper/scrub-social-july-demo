# Web UI Kit — Scrub Society (scrubsociety.com)

Marketing homepage for Scrub Society. Flow:

1. **Nav** — sticky, frosted, with Join CTA
2. **Hero** — "Belong everywhere." headline + editorial art block
3. **Strip** — four brand stats (members / partners / pay / states)
4. **Pillars** — six JTBD cards (find a job, housing, community, pathways, travel support, share)
5. **Spotlight** — editorial grid of member features
6. **Join** — Lavender Field banner with "Flourish in your field."
7. **Footer** — inverted Plum Grove footer

Click any **Join** CTA to open the sign-up modal.

## Placeholders

- Hero art and spotlight imagery are decorative color washes (Lavender Field / Open Air / Clear Sky). Replace with real editorial photography from the brand shoot.
- Pillar counts, partner counts, and editorial copy are reasonable seed content. Swap with real copy from the content team.
